# CLI 开发
CLI 工具开发将使用 TS 作为开发语言，如果此时还没有接触过 TS 的同学，刚好可以借此项目来熟悉一下 TS 的开发模式。   

```
    mkdir cli && cd cli // 创建仓库目录
    npm init // 初始化 package.json
    npm install -g typescript // 安装全局 TypeScript
    tsc --init // 初始化 tsconfig.json
```
